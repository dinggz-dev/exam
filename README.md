# exam
考试系统

## 一、准备工作
1.安装jdk1.8、eclipse、mysql（5.7以上版本），配置maven
2.配置application.properties中的数据库连接信息
3.新建数据库exam
4.直接InitSystem中的InitSystem方法，初始化系统
